# Tienda de ropa (SSR + Dashboard + MongoDB Atlas)

Proyecto backend con Express + Mongoose.
Renderiza HTML con template literals (SSR), incluye dashboard protegido por login y una API JSON.

## Tecnologías
- Node.js + Express
- MongoDB Atlas + Mongoose
- method-override (PUT/DELETE desde formularios)
- express-session (login)
